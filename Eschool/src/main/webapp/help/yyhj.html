三、课件运行环境

电脑软、硬件要求：
　　 奔腾以上CPU、32M以上内存、56K MODEM、声卡、音箱、能够连接到互联网。 　　
　　 软件：操作系统：Windows98第二版以上；建议Windows 2000或者Windows XP加IE6。注意：如果使用WindowsNT，须安装Service Pack 6以上；浏览器版本：IE 6.0或以上；Windows Media player7.1或以上。
　　
　　 课件浏览器要求： Internet Explore（windows操作系统自带）
　　 一般windows98、2000系统用户自带的IE浏览器都是低于6.0版本的，为了保证能正常浏览课件，建议最好是安装Internet Explore6.0或以上版本，进入此页面下载Internet Explore6.0。

　　 课件播放器要求：
　　 网络课堂内讲座的音频和视频采用的是Windows Media player播放器,一般windows操作系统用户不用再安装任何播放器就可直接播放。一般进入课堂后，音频讲座会自动播放，但视频讲座必须安装Windows Media Player7.1或以上版本的播放器才能正常播放,而Windows 98 SE、2000的系统用户自带的Windows Media Player都低于7.1的，所以请Windows Media Playe低版本的用户进入此页下载适合本机系统的Windows Media Player播放器。
